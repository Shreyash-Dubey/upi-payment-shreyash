# UPI Payment App (Vite + React + Tailwind)

## What you have
- React + Vite starter
- Tailwind CSS configured
- lucide-react icons installed
- Split components (Header, SendMoney, History, UserProfile, Modals)
- Dummy transactions and mock PIN flow

## Quick start (after download)
1. Install Node.js (LTS) from https://nodejs.org/
2. In project folder run:
   ```bash
   npm install
   npm run dev
   ```
3. Open the URL shown by Vite (usually http://localhost:5173)

## Notes
- This is a demo app. Do **not** store real PINs or secrets in code.
- To enable production-ready features (real API, auth, encryption), integrate a backend.
